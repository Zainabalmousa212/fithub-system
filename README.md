# FitHub – Gym Management System

FitHub is a full-stack gym management system that helps gyms manage members, trainers, and workouts through a modern web interface.

## Tech Stack

Frontend
- React
- Vite
- TypeScript
- TailwindCSS

Backend
- Python
- Flask
- JWT Authentication
- SQLite

## Features

- User authentication
- Member management
- Trainer dashboard
- REST API backend
- Responsive UI

## Running the Project

### Backend

cd backend  
pip install -r requirements.txt  
python app.py

### Frontend

cd frontend  
npm install  
npm run dev
